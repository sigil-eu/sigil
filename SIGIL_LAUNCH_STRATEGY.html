1. Hacker News ‚Äî "Show HN" Post

Highest Priority

Title

Show HN: SIGIL ‚Äì The missing security layer for MCP agent-to-tool interactions

Post Body

MCP (Model Context Protocol) lets AI agents call tools, but has zero security:
no identity verification, no content scanning, no audit trail, no permission
gating.

SIGIL is an open Rust protocol that fills this gap. 5 traits you implement
with your own backends:

  - IdentityProvider ‚Äî bind users to trust levels (eIDAS, OIDC, SSI)
  - SensitivityScanner ‚Äî detect secrets before they reach the LLM
  - VaultProvider ‚Äî encrypt intercepted content
  - AuditLogger ‚Äî tamper-evident event logging
  - SecurityPolicy ‚Äî risk gating + rate limiting

Plus a reference MCP server that wraps any tool set with all five layers.

  cargo add sigil-protocol
